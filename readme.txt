== Description ==

It makes possible to hide or to show the arbitrary blocks of text from the registered users, guests on the site or the search robots

== Installation ==

1. Unzip archive and upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to /wp-admin/options-general.php?page=hidden-content/hidden-content.php page for congigurate
